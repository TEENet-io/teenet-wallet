# 审批与 Web 界面

[TEENet Wallet][wallet-url] 的 Web 界面是你管理钱包、查看审批和监控活动的控制面板。日常操作主要由 Agent 处理，Web 界面则用于监督和安全配置。

顶部导航有三个标签页：**钱包**、**审批**、**活动**。齿轮图标打开 **设置**。

---

## 钱包

**钱包** 标签页会显示你的所有钱包。每张卡片都会显示钱包名称、链、地址、余额和状态。

**创建钱包：** 点击顶部的 **Create New Wallet**，从下拉列表选择链，可选填写标签，然后点击 **Create Wallet**。EVM 钱包需要 1 到 2 分钟完成分布式密钥生成；Solana 钱包通常即时完成。

**钱包详情页：** 点击任意钱包进入详情页，可查看：

- **余额**：当前余额和对应的 USD 估值
- **Threshold** 标签页：配置消费策略
  - **Approval threshold (USD)**：超过该金额的交易需要你的 Passkey
  - **Daily limit (USD)**：每日总支出的硬上限，进度条会显示当天已花费金额
  - 点击 **Save policy** 应用策略（需要 Passkey 确认）
- **Contract** 标签页（Solana 上显示为 **Program**）：查看和管理白名单合约。只有白名单中的合约或程序才能被调用。添加或删除都需要 Passkey。
- **Delete Wallet**：永久删除钱包（需要 Passkey）

---

## 审批

**审批** 标签页会显示所有等待你处理的交易。每条待审批记录都会显示：

- 金额和币种（例如 **0.001 ETH**）
- 接收方地址
- **PENDING** 状态标记
- 距离过期还剩多少时间的倒计时

**批准：** 点击一条待审批记录，检查详情后点击 **Approve**。然后使用指纹或面容识别完成验证。验证成功后，交易会被签名并发送到链上。

**拒绝：** 如果内容看起来不对，点击 **Reject**。该交易会被取消。

**过期：** 待审批请求默认 **24 小时** 过期。到期未处理会自动取消。如果你仍然想继续，告诉 Agent 重新发起即可。

**入口：** 当交易需要你确认时，Agent 会直接发给你审批页面链接。你也可以随时登录 Web 界面，在 **审批** 标签页中查看。

---

## 活动

**活动** 标签页会按日期展示账户上的完整审计记录。每条记录都会显示动作类型、相关细节和时间戳。常见类型包括：

- **Transfer**：转账，带有 **AUTO**（自动执行）或 **PENDING**（等待审批）状态
- **Approve**：你批准了一条待审批请求
- **Policy Update**：审批阈值或日限额发生变化
- **Contract Call**：智能合约交互
- **Login**：账户登录事件

你可以使用 **All Actions** 下拉菜单按动作类型筛选，也可以点击 **Refresh** 加载最新记录。

---

## 设置

点击齿轮图标打开 **Security & API Control**：

- **API Keys**：生成新密钥（输入标签后点击 **Generate**），查看已有密钥的创建时间和状态，或通过撤销图标废止某个密钥。每个密钥会显示部分遮罩前缀（例如 `ocw_84b888...`）。
- **Address Book**：把地址保存成昵称，方便快速转账。点击 **Add Address** 可创建新条目。
- **Danger Zone**：
  - **Logout Global**：让所有会话退出登录
  - **Delete Account**：永久删除你的账户和所有数据
