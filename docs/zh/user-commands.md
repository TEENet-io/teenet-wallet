# 你能做什么

你通过和 AI Agent 对话来使用钱包。直接用自然语言告诉它你想做什么即可。

---

## 钱包与余额

- **"创建一个叫 Trading 的以太坊钱包"**
- **"显示我的钱包"**：列出所有钱包、地址、链和标签
- **"我的余额是多少？"**：查看钱包余额

## 转账与 DeFi

- **"转 0.1 ETH 到 0xABC...123"**：如果低于你的阈值，会直接发送
- **"转 0.1 ETH 到 alice"**：发送到地址簿中保存为 `alice` 的地址
- **"发送 50 USDC 到 0xDEF...456"**：ERC-20 代币转账；Agent 会确保合约已在白名单中
- **"在 Uniswap 上把 0.5 ETH 换成 USDC"**：Agent 会构建交易并发给你审批链接，所有合约交互都需要 Passkey

## 设置与安全

- **"把审批阈值设置为 200 美元"**：修改消费阈值（需要 Passkey）
- **"把 USDC 合约加入白名单"**：发起合约白名单提案（需要 Passkey）
- **"显示待审批交易"**：查看等待你确认的交易
- **"今天花了多少？"**：查看今日 USD 支出和剩余日预算
- **"把 alice 保存为 0xABC...123（Ethereum）"**：把联系人保存到地址簿

---

## 哪些操作需要审批？

大多数只读操作，例如查余额、历史记录和价格，都不需要审批。低于 USD 阈值的转账会自动执行。

以下操作需要 Passkey：

- 超过阈值的转账
- 所有智能合约交互，例如 swap、代币授权和 DeFi 操作
- 白名单、策略和 API 密钥相关变更

超过日限额的转账会被直接拒绝，直到额度在 UTC 午夜重置。审批页面和账户控制说明见 [审批与 Web 界面](/zh/user-approvals.md)。

## 快捷命令

部分 Agent 平台支持快捷命令。例如在 [OpenClaw](https://openclaw.ai) 中：

| 命令 | 功能 |
|---------|-------------|
| `/balance` | 显示所有钱包余额 |
| `/transfer 0.1 ETH to 0xabc...` | 发送加密货币 |
| `/approve` | 查看待审批请求 |
| `/whitelist` | 查看白名单合约 |
| `/policy` | 查看当前审批策略 |
| `/contacts` | 查看地址簿 |
| `/spent` | 今日 USD 支出 |
| `/chains` | 可用区块链 |

你也可以直接用自然语言表达同样的意思。
