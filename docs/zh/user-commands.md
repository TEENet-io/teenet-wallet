# 和 OpenClaw 对话

这是你使用 TEENet Wallet 的主要方式。你可以用自然语言或快捷命令，效果一样。

---

## 功能总览

| 功能 | 说明 | 需要审批？ |
|------|------|-----------|
| **创建钱包** | 在以太坊、Solana 等链上创建钱包 | 否 |
| **查看余额** | 查看所有钱包的原生代币和代币余额 | 否 |
| **转账** | 发送 ETH、SOL、ERC-20 或 SPL 代币 | 超过 USD 阈值时需要 |
| **DeFi 交易** | 与 Uniswap 等白名单合约交互 | 是（所有合约调用） |
| **代币授权** | 授权或撤销第三方代币支出权限 | 是（始终） |
| **Wrap / Unwrap SOL** | 原生 SOL 与 wSOL 互转 | 超过阈值时需要 |
| **读取合约数据** | 查询链上余额、授权额度、价格等 | 否 |
| **管理白名单** | 添加或删除白名单合约 | 添加需要；删除仅 Passkey |
| **设置消费策略** | 配置 USD 阈值和日限额 | 是 |
| **查看审批** | 查看和处理待审批请求 | 否 |
| **查看历史** | 查看所有操作的审计记录 | 否 |
| **查看币价** | 查看 ETH、SOL 和稳定币价格 | 否 |
| **管理 API 密钥** | 生成或撤销 API 密钥 | 仅 Passkey |

## 自然语言示例

### 钱包管理

- **"创建一个叫'交易用'的以太坊钱包"** -- OpenClaw 会创建钱包并返回地址
- **"显示我的钱包"** -- 列出你所有钱包的地址、链和名称
- **"我的余额是多少？"** -- 查询当前钱包的链上余额
- **"显示所有钱包余额"** -- 一次性查看所有钱包的资产情况

### 发送加密货币

- **"转 0.1 ETH 到 0xABC...123"** -- 发送以太坊原生代币
- **"发送 50 USDC 到 0xDEF...456"** -- 发送 ERC-20 代币
- **"转 1 SOL 到 ABC...XYZ"** -- 发送 Solana 原生代币

如果转账金额低于你设定的阈值，OpenClaw 会即时完成并告诉你交易哈希。如果超过阈值，OpenClaw 会在聊天中发给你一个审批链接，等你用指纹确认后再执行。

### DeFi 和合约交互

- **"在 Uniswap 上把 0.5 ETH 换成 USDC"** -- OpenClaw 构建交易并发给你审批链接。所有合约操作都需要你用 Passkey 确认
- **"授权 Uniswap 路由器使用我的 USDC"** -- 代币授权操作（需要审批）
- **"查询 USDC 合约上我的余额"** -- 读取合约数据，无需审批

合约交互需要目标合约在白名单中。如果合约不在白名单里，OpenClaw 会自动发起添加提案，你在审批中确认即可。

### 白名单管理

- **"把 USDC 合约加到白名单"** -- OpenClaw 发起添加提案
- **"显示白名单里的合约"** -- 查看已授权的合约列表

注意：添加白名单需要你在 Web 界面或通过审批链接用 Passkey 确认。

### 策略和安全

- **"设置审批阈值为 200 美元"** -- 调整消费限额
- **"显示我的审批策略"** -- 查看当前阈值和日限额设置
- **"显示待审批的交易"** -- 查看等待你确认的交易
- **"今天花了多少？"** -- 查看今日 USD 支出和剩余额度

## 快捷命令

熟练用户可以用这些快捷方式提速：

| 命令 | 功能 |
|------|------|
| `/balance` | 显示所有钱包余额 |
| `/balance eth` | 显示指定链的余额 |
| `/wallets` | 列出所有钱包 |
| `/transfer 0.1 ETH to 0xabc...` | 转账 |
| `/approve` | 查看待审批请求 |
| `/approve 123456` | 查看指定审批详情 |
| `/whitelist` | 查看白名单合约 |
| `/whitelist 0xabc...` | 添加合约到白名单 |
| `/policy` | 查看当前审批策略 |
| `/policy 100` | 设置阈值为 $100 |
| `/spent` | 今日已花费和剩余额度 |
| `/prices` | 当前币价 |
| `/chains` | 支持的区块链列表 |
| `/call 0xabc... method(args)` | 调用智能合约 |

命令不区分大小写。你也可以用自然语言说同样的事情，OpenClaw 都能理解。
