# 简介

## 什么是 TEENet Wallet？

TEENet Wallet 是一款多链加密货币钱包，私钥永远不会完整存在于任何一台机器上。每一把私钥通过门限密码学分散到多个可信执行环境（TEE）硬件节点中。签名时，多个 TEE 节点协同完成 -- 完整私钥从未被还原。

安全性媲美硬件钱包，使用体验和普通 API 一样简单。

---

## 为什么选择 TEENet Wallet？

| 传统钱包 | TEENet Wallet |
|---|---|
| 私钥存在单一位置 | 密钥分片分布在 TEE 节点 |
| 单点攻破即失控 | M-of-N 门限 -- 单节点无法签名 |
| 手动审批或无审批 | 可配置 USD 阈值 + Passkey 硬件审批 |
| 一次只支持一条链 | 以太坊、Solana 及所有 EVM 链一个 API 搞定 |

---

## 核心功能

- **多链支持** -- 以太坊、Solana 及任意 EVM 兼容链，支持运行时动态添加自定义链
- **双重认证** -- API Key 供 Agent 和自动化使用，Passkey 供人类审批敏感操作
- **USD 消费控制** -- 每个钱包可设阈值和日限额，稳定币按 $1 计价，其他币种通过 CoinGecko 实时换算
- **合约白名单** -- 仅预先批准的合约可被调用，所有合约操作通过 API Key 发起时需人工审批
- **代币转账** -- ERC-20 和 SPL 代币由后端构建、签名、广播，Solana ATA 自动创建
- **完整审计** -- 每一笔操作记录认证方式、API Key 标签和时间戳

## 签名流程

```
 AI Agent / 应用                       用户（浏览器）
     |  API Key (ocw_*)                    |  Passkey (ps_*)
     v                                     v
+--------------------------------------------------------------+
|  TEENet Wallet  (:8080)                                      |
|  REST API · 审批策略 · 合约白名单                               |
+--------------------------------------------------------------+
     |  TEENet SDK
     v
+--------------------------------------------------------------+
|  app-comm-consensus  (:8089)                                 |
|  M-of-N 投票协调                                              |
+--------------------------------------------------------------+
     |  gRPC + mTLS
     v
+--------------------------------------------------------------+
|  TEE-DAO 密钥管理集群（3-5 节点）                               |
|  门限签名 · 私钥永远不离开 TEE 硬件                              |
+--------------------------------------------------------------+
```

1. 客户端发送请求（API Key 或 Passkey）
2. 钱包检查白名单、阈值和日限额
3. 需要审批时，请求进入待审批状态，等待所有者用 Passkey 确认
4. 钱包将签名请求路由到 TEE 集群
5. TEE 节点协作生成门限签名 -- 完整私钥从未被还原
6. 钱包广播已签名交易并返回交易哈希
