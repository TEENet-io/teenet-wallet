# 审批与 Web 界面

当 OpenClaw 发起超过阈值的交易、与新合约交互或执行其他敏感操作时，交易不会立刻执行，而是等待你的审批。

---

## 审批交易

**你会在聊天中收到通知。** OpenClaw 会告诉你有一笔交易需要你确认，并附上审批链接。

**打开审批链接。** 点击 OpenClaw 发给你的链接，会直接跳转到该笔交易的审批页面。你也可以登录 [TEENet Wallet][wallet-url]，在 **Approvals**（审批）标签页查看所有待处理的审批请求。

**审批页面会显示交易详情：** 操作类型、金额、代币种类、发送方钱包、接收方地址、请求时间和剩余有效时间。仔细检查这些信息，确认是你授权 OpenClaw 执行的操作。

**批准或拒绝：**
- 点击 **Approve**（批准）并用指纹或面容验证，交易会自动签名并广播到区块链
- 点击 **Reject**（拒绝）取消交易，OpenClaw 会收到拒绝通知

<div align="center"><img src="picture/appqueue.png" alt="审批详情页" width="360" /></div>

**审批有效期为 24 小时。** 超时未处理的请求会自动过期。如果错过了审批窗口，告诉 OpenClaw 重新发起操作即可。

## Web 界面功能概览

Web 界面（[TEENet Wallet][wallet-url]）是你管理钱包和监督 OpenClaw 的控制面板：

- **钱包** -- 查看所有钱包的地址、链、余额。点击钱包进入详情页查看消费策略和合约白名单
- **审批** -- 在审批标签页处理 OpenClaw 发起的待审批请求
- **活动** -- 查看完整的审计记录，包括每一笔转账、审批、策略变更，支持按操作类型筛选
- **设置**（齿轮图标）-- 管理 API 密钥、地址簿、删除账户
- **审批策略** -- 在钱包详情页的 **阈值** 标签页设置审批阈值和日上限
- **合约白名单** -- 在钱包详情页的 **合约** 标签页（Solana 为 **程序**）管理白名单，添加和删除需要 Passkey 验证
- **地址簿** -- 在设置页管理常用地址，通过昵称快速转账

日常使用中，你主要通过和 OpenClaw 聊天来操作钱包。Web 界面更多用于审批交易、调整安全设置和查看历史记录。
